## 概要
Django用のSimpleなCMSアプリケーションです。
Djangoを利用してシステムを構築するエンジニア向けに、以下の3つをコンセプトで作成しました。
* 簡単に使える
* 簡単に理解できる
* 簡単に拡張できる

DjangoでパーツとしてCMSを利用したい時や、CMSの勉強に利用できます。
